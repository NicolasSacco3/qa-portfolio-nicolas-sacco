API Testing – Demoblaze
🧾 Descripción

Proyecto de testing de API realizado sobre la plataforma Demoblaze utilizando Postman.

Se validaron endpoints críticos relacionados con autenticación, productos y flujo de compra, verificando respuestas, códigos de estado y estructura del JSON.

🎯 Objetivos

Validar endpoints principales del sistema

Comprobar códigos de estado HTTP

Verificar estructura y contenido del response

Probar escenarios positivos y negativos

Validar autenticación y manejo de errores

📊 Alcance

✔ Pruebas de autenticación
✔ Validación de productos
✔ Validación de carrito
✔ Pruebas con datos válidos e inválidos
✔ Verificación de status codes (200, 400, 401, etc.)

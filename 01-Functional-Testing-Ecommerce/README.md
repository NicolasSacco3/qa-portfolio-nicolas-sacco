📌 Proyecto: Testing Funcional – Plataforma E-commerce

🧾 Descripción

Proyecto de testing manual realizado sobre la plataforma Demoblaze, simulando un entorno real de trabajo QA.

Se diseñaron y ejecutaron casos de prueba funcionales, documentando defectos encontrados y evidencias de ejecución.

🎯 Objetivos del Proyecto

Validar el flujo completo de compra

Verificar comportamiento en Web y Mobile

Detectar defectos funcionales y de interfaz

Evaluar validaciones del formulario de compra

Comprobar restricciones de acceso (login obligatorio)
